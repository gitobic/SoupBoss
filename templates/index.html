CLI Commands for Bulk Operations

For bulk operations and automation, use these CLI commands:

Bulk Company Management
# Add multiple companies individually
uv run python main.py companies add spacex --source greenhouse
uv run python main.py companies add leverdemo --source lever
uv run python main.py companies add dynatrace1 --source smartrecruiters
Bulk Job Fetching
# Fetch from companies file (one company per line)
uv run python main.py jobs fetch --source greenhouse --companies-file companies.txt

# Fetch jobs for specific companies
uv run python main.py jobs fetch --source greenhouse --company spacex
uv run python main.py jobs fetch --source lever --company leverdemo
Complete Automation Workflow
# 1. Add multiple companies and fetch all jobs
uv run python main.py companies add company1 --source greenhouse
uv run python main.py companies add company2 --source lever
uv run python main.py jobs fetch --source greenhouse --company company1
uv run python main.py jobs fetch --source lever --company company2

# 2. Process and analyze
uv run python main.py match generate
uv run python main.py match run
uv run python main.py match export --format csv --output results.csv
Tip: Create a companies.txt file with one company name per line, then use --companies-file companies.txt for efficient bulk processing.
